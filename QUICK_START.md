## 📖 快速开始

### 1️⃣ 启动应用
```bash
cd c:\Users\Shaolou\AI_Patent
streamlit run app.py
```

### 2️⃣ 输入想法
在界面中填写：
- **发明名称**：给你的发明起个名字
- **技术描述**：描述你的技术方案（越详细越好）

- 查看评分（新颖性、创造性等）
- 切换Tab查看不同部分
- 在线编辑内容

点击 **"📤 导出专利"**，下载Markdown或Word文档

## 💡 核心改进

### 1. 端到端生成
不再分步骤调用LLM，一次性生成：
- ✅ 创新点分析
- ✅ 五要素分析
- ✅ 专利摘要
- ✅ 权利要求书
- ✅ 说明书
- ✅ 新颖性评估

### 2. 智能默认值
- 自动推断应用场景
- 智能格式化技术描述
- 默认质量级别为"标准"

### 3. 质量控制
支持三种质量级别：
- **基础**：快速生成，满足基本要求
- **标准**：符合专利法要求 ⭐推荐
- **高质量**：专业级专利文档

## 📝 输入建议

### 好的技术描述示例
```
开发了一种基于注意力机制和多尺度特征融合的图像识别方法，
通过引入轻量级卷积块和特征金字塔网络，实现了在保持高精度的
同时大幅降低计算复杂度。该方法适用于移动端实时图像处理场景。
```

### 应该包含的信息
1. **核心技术原理** - 你的技术是怎么工作的
2. **关键技术特征** - 相比现有技术的独特之处
3. **预期技术效果** - 能达到什么性能指标
4. **解决的技术问题** - 解决了什么痛点

## ❓ 常见问题

### Q: 生成失败怎么办？
A: 检查：
1. API Key是否配置正确
2. 网络连接是否正常
3. 技术描述是否太短

### Q: 生成的内容不满意？
A: 可以：
1. 在审查页面编辑内容
2. 调整质量级别重新生成
3. 添加参考专利后再生成

### Q: 如何保存我的专利？
A: 点击 **"📤 导出专利"**，下载为：
- Markdown格式（方便分享）
- Word格式（方便打印提交）
